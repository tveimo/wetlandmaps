Widget Help: Maps Online Report Request Popup

Introduction

Welcome to the help page for the Maps Online Report Request Popup widget. This tool enhances your Experience Builder mapping applications by allowing users to submit requests for Maps Online Reports. When a user selects a feature on a map, a button will appear in the popup to launch a request form. This guide provides instructions to help administrators configure the widget effectively.

Configuration Steps

Follow these steps to ensure your widget functions correctly:

  • Drag and drop the widget somewhere unobtrusive onto your page layout, such as a bottom corner. Do not place the widget inside a Widget Controller.
  • Note: the widget will appear as a small icon in the published application. The end user will not be able to interact with this icon, it simply lets the widget run in the background.
  • Once you have added the widget to your page, click on the widget to open the configuration panel.
  • Under the Source setting, use the drop down box to select the Map component the widget will interact with.
  • Fill in the Application Name setting with your application name as configured by the Maps Online system. If you are unsure what this is, please contact the Science Division's Spatial Team.
  • Leave the Advanced Options settings toggled off.
  • Be sure to click the Save button to save your widget settings.
  • Note: you will need to enable popups for any map layers that you wish to have this widget's functionality. The popup settings are configured in your ESRI Portal's Map Viewer.
