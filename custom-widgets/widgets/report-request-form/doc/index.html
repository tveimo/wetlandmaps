Widget Help: Maps Online Report Request

Introduction

Welcome to the help page for the Maps Online Report Request widget. This tool enhances your Experience Builder mapping applications by allowing users to submit requests for Maps Online Reports. With an interactive form, users can draw their area of interest on the map and choose the specific report they need. This guide provides instructions to help administrators configure the widget effectively.

Configuration Steps

Follow these steps to ensure your widget functions correctly:

  • Drag and drop the widget into a Widget Controller in your page layout. For more information on Widget Controllers see here.
  • Click on the widget to open the configuration panel.
  • Under the Source setting, use the drop down box to select the Map component the widget will interact with.
  • Fill in the Application Name setting with your application name as configured by the Maps Online system. If you are unsure what this is, please contact the Science Division's Spatial Team.
  • Leave the Advanced Options setting toggled off.
  • Be sure to click the Save button to save your widget settings.
